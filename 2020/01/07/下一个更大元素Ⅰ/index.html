题目：下一个更大元素Ⅰ（使用解法有暴力解法和单栈解法)

题目描述：

给定两个没有重复元素的数组 nums1 和 nums2 ，其中nums1 是 nums2 的子集。找到 nums1 中每个元素在 nums2 中的下一个比其大的值。

java

示例：

java

注意:

  1. nums1和nums2中所有元素是唯一的。
  2. nums1和nums2 的数组大小都不超过1000。

解题思路一：

暴力法思路:

解决此题目的关键点应该在于nums1是nums2的子集，那么nums2的长度就必定比nums1大

  • 设定一个栈将nums2从后到前压入栈，那么这样最后面的数字就会最后弹出来 再设定一个temp指针指向nums1中正在对比的元素
  • 当nums2的弹出次数少于等于count时不做比较
java

解题思路二：

单栈解法：

我们首先把第一个元素 nums2[1] 放入栈，随后对于第二个元素 nums2[2]，如果 nums2[2] > nums2[1]，那么我们就找到了 nums2[1] 的下一个更大元素 nums2[2]，此时就可以把 nums2[1] 出栈并把 nums2[2] 入栈；如果 nums2[2] <= nums2[1]，我们就仅把 nums2[2] 入栈。对于第三个元素 nums2[3]，此时栈中有若干个元素，那么所有比 nums2[3] 小的元素都找到了下一个更大元素（即 nums2[3]），因此可以出栈，在这之后，我们将 nums2[3] 入栈，以此类推。

java

总结

单栈法相对难以理解需要自己多画几次图或者结合debug思考。主要突破点为nums1为nums2的子集所以只需要找到nums2每个元素的目标值即可，获得的方法为遍历，没有找到的压入栈，直到找到，最终在栈底的元素是为无目标值。
