题目：删除最外层的括号（可用栈可不用）

题目描述：

有效括号字符串为空 (“”)、”(“ + A + “)” 或 A + B，其中 A 和 B 都是有效的括号字符串，+ 代表字符串的连接。例如，””，”()”，”(())()” 和 “(()(()))” 都是有效的括号字符串。

如果有效字符串 S 非空，且不存在将其拆分为 S = A+B 的方法，我们称其为原语（primitive），其中 A 和 B 都是非空有效括号字符串。

给出一个非空有效字符串 S，考虑将其进行原语化分解，使得：S = P_1 + P_2 + … + P_k，其中 P_i 是有效括号字符串原语。

对 S 进行原语化分解，删除分解中每个原语字符串的最外层括号，返回 S 。

示例：

java

注意:

  • S.length <= 10000
  • S[i] 为 "(" 或 ")"
  • S 是一个有效括号字符串

解题思路一（使用栈）：

有点类似于《有效的括号》，思路主要为设定四个参数，一个boolean判定是否是原语，一个stack储存“（”，遇到“）”则出栈，int值start和end根据end和start判断当前原语的初始和结束位置

java

解题思路二（不使用栈）：

主要目的是找到每个原语的start和end索引，所以可以不用stack作为容器，添加一个int参数取代stack判断原语是否结束的功能。

java

总结

两种解法都是看了答案的提示，看来对于栈的理解还不够。
