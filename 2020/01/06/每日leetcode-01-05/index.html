题目：最小栈（此题目的难点在于添加了一个getMin的操作)

题目描述：设计一个支持 push，pop，top 操作，并能在常数时间内检索到最小元素的栈。

java

示例:

MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.getMin(); –> 返回 -3.
minStack.pop();
minStack.top(); –> 返回 0.
minStack.getMin(); –> 返回 -2.

解题思路一：

使用数组的方式完成模拟栈，设置一个尾指针temp。需要考虑扩容/空判断的情况

java

解题思路二：

使用栈类来解答该问题，因为栈不能遍历，所以需要设置一个辅助栈（helper），辅助栈和主栈需要一起参与pop和push操作：这一步时为了当一个Min值在主栈被取出时，辅助栈中也会取出相应的Min值。

java
